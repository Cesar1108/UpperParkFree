The files in this folder provide a base color syntax highlighing and code/region folding for the following file types.
To use any of these copy the files to:
%Appdata%\SoftVelocity\Clarion\8.0\modes folder and re-start Clarion (Change 8.0 to correct version of Clarion)

Clarion-Included-Projects-Mode.xshd
	Syntax Highlighting for .PR and .PRJ files

BAT-Mode.xshd
	Syntax Highlighting for .BAT and .CMD files

INI-Mode.xshd
	Syntax Highlighting for .INI files

Clarion-RED-Mode.xshd
	Syntax Highlighting for .RED files
